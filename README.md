# Estudo de POO em Java

Projeto criado para praticar conceitos de Programação Orientada a Objetos em Java.

## 🚀 Tecnologias
- Java

## 📚 Conceitos aplicados
- Encapsulamento
- Herança
- Polimorfismo

## ▶️ Como executar
1. Clone o repositório
2. Compile:
   javac Main.java
3. Execute:
   java Main
